localStorage and sessionStorage are almost the same. Both have the following rules:
They store key/value pairs for a particular domain. The values stored may only be strings. They have a limit of around 5MB of storage space.
-localStorage persists the storage across browser sessions. & the sessionStorage is cleared when the page session ends.
-while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends.

Function Scope: When a variable is declared inside a function, it is only accessible within that function and cannot be used outside that function. Block Scope: A variable when declared inside the if or switch conditions or inside for or while loops, are accessible within that particular condition or loop.

Event loop is an endless loop, which waits for tasks, executes them and then sleeps until it receives more tasks. The event loop executes tasks from the event queue only when the call stack is empty i.e. there is no ongoing task. The event loop allows us to use callbacks and promises.
